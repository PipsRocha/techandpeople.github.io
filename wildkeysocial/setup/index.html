Download, install and set up WildKey Social
After downloading the apk file, you will need to grant the permission to "Install unkown apps" to your browser. Normaly the phone asks for this when you click on the downloaded file. If you are not prompt to grant this permission, go to settings > applications > your browser name > Install unkown apps.
Example with vivaldi, in your phone it migh be chrome or other browser.
Setting Up WildKey Social
1. Disable battery saver after instalation (Xiaomi as example, other brands should be similar)
2. Start the process
3. Grant permissions
4. Insert the provided code
5. Activate the Service
6. In the end you should end up with this screen and receive this notification.
