WildKey
A privacy-aware keyboard toolkit for data collection in-the-wild
With our keyboard you will be able to collect everyday typing data enabling continuous and spontaneous assessments of motor skills. Try out our demo (Android 6.0+).
Data Collections
Implicit Typing Collection
Explicit Typing Collection
Send out text-entry tasks for your users to perform.
Questionnaires
Create and schedule questionnaires for your study participants.
Customizable tasks
Support to create and use custom made tasks for your study. Currently available: Alternate Finger-Tapping.
Privacy-aware
No raw text is collected!
By default, the Implicit method the keyboard analysys and calculates all metrics locally on the keyboard before sending out only the results of the analysys (performance and behaviors metrics) to the databases.
Private mode
At any point users can select the lock always available when the keyboard is opened to enter the “private mode” which disables all text analytics. No data is processed or store while the user remains in private more.
Study management
You have the ability to schedule tasks (questionnaires, explicit typing collection and other) to a specific user or to groups. Users will receive a notification of pending tasks, have a reminder shown on the keyboard, and in addition they are also able to access it by opening the keyboard dedicated app.
